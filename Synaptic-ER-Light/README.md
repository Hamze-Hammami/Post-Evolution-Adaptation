This repository contains evaluation code that applies Hebbian learning to pre-trained genotypes 
usuing evltionary robotics to study and research the effects of synaptic and stimuli reactions in a robotic system.

# to fully vislaise the effect of Hebbian on gene follow these steps 

1- run the robot wuth original genes by pressing 'R' where lumonisity on background is set to 1 on both background lights 
2- change lumonisity on background to 0.1 on both background lights 
3- run the robot wuth original genes again, this will fail 
4- finally run the robot with synaptic on a Hebbian rate = 0.002 this should already be set but if not change it in controller 
5- robot will adapt to new light enviroment
